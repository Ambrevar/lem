# lem
![screenshot](./screenshots/screenshot.png)

## Requirement
- https://github.com/roswell/roswell
- ncurses

## Installation
`$ sudo apt install libncurses5-dev  # for debian`  
`$ ros install cxxxr/lem`

## Usage
`$ lem`  

    CL-USER> (ql:quickload :lem)
    CL-USER> (lem:lem)
