## Requirement
- [roswell](https://github.com/roswell/roswell)
- ncurses

## Installation
`$ ros install cxxxr/lem`

## Usage
`$ lem`  

    CL-USER> (ql:quickload :lem)
    CL-USER> (lem:lem)
